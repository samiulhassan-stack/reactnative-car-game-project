QUICK START TROUBLESHOOTING
================================================================================

Issue: "npm: command not found"
→ Install Node.js from https://nodejs.org/

Issue: "Cannot find adb"
→ Set ANDROID_HOME environment variable

Issue: No emulator available
→ Create one in Android Studio (Tools > AVD Manager)

Issue: Build fails
→ cd android && gradlew clean && cd ..

Issue: App crashes
→ adb shell pm clear com.carracing.game && npm run android

See GETTING_STARTED.md or COMPLETE_GUIDE.md for more help.
