## 🔧 Troubleshooting Quick Ref

| Issue | Solution |
|-------|----------|
| Build takes too long | First build is normal (3-5 min). Use cached builds after. |
| Emulator slow | Close other apps. Increase RAM in emulator settings. |
| Build fails | Restart emulator and clear cache: `npm start -- --reset-cache` |
| Port conflict | Kill node: `taskkill /F /IM node.exe` |
| Any other issue | See COMPLETE_GUIDE.md |
